## 2026-07-10 - Enhanced Accessibility in HTML Documentation

**Learning:** Proper semantic HTML (like `<thead>`, `<tbody>`, and `<th scope="col">`) is required for tables, and ARIA live regions (`role="alert"` and `role="status"`) should be applied to visual callouts so screen readers can relay information properly to users without sight.
**Action:** Always verify semantic structure and ARIA attributes for non-interactive content blocks like tables and alerts when making HTML enhancements.
