⚡ Claude Gateway

Proxy local para o Claude for Office Add-in

v1.0 — Guia de Instalação

0 Pré-requisitos

  • Windows 10/11 ou macOS 12+
  • Microsoft Word, Excel ou PowerPoint (desktop)
  • Acesso ao Word Online, Excel Online ou PowerPoint Online para carregar o manifest
  • Conta gratuita no ZenMux.ai — copie sua API Key
O pacote já inclui Node.js e Python embutidos.
Não é necessário instalar nada além de extrair os arquivos.

1 Extrair o pacote

  1. Baixe o arquivo claude-gateway-v1.0.zip
  2. Extraia na pasta do seu sistema:
C:\Users\SEU_USUARIO\.office-addin-dev-certs\
~/.office-addin-dev-certs/

Estrutura esperada

📁 .office-addin-dev-certs/
├── 00 - Comece Aqui.html  ← Este guia
├── Claude Gateway.vbs  ← Lançador (Windows)
├── localhost.crt  ← Certificado SSL
├── manifest.xml  ← Manifesto do Add-in
└── 📁 .app/  ← Pasta oculta (não modificar)
    ├── proxy-cors.js
    ├── gateway_gui.py
    ├── gateway-config.json
    ├── 📁 node/  ← Node.js embutido
    └── 📁 python/  ← Python embutido

2 Instalar o Certificado SSL

O proxy usa HTTPS local. O certificado localhost.crt precisa ser instalado como confiável.

  1. Dê duplo-clique no arquivo localhost.crt (na raiz da pasta)
  2. Clique em Instalar Certificado…
  3. Selecione Máquina Local → Avançar
  4. Selecione Colocar todos os certificados no repositório a seguir
  5. Clique Procurar… → Autoridades de Certificação Raiz Confiáveis
  6. Confirme com OK → Concluir
  1. Dê duplo-clique no arquivo localhost.crt
  2. O Acesso às Chaves (Keychain Access) abrirá
  3. O certificado será adicionado ao chaveiro login
  4. Localize o certificado localhost no chaveiro
  5. Dê duplo-clique → expanda Confiança
  6. Em "Ao usar este certificado", selecione Confiar Sempre
  7. Feche e confirme com sua senha
Sem esse passo, o Word/Excel/PowerPoint rejeitará a conexão HTTPS com o proxy local.

3 Carregar o Add-in via Office Online

Acesse qualquer um dos aplicativos online da Microsoft:

  • word.cloud.microsoft.com — para Word
  • excel.cloud.microsoft.com — para Excel
  • powerpoint.cloud.microsoft.com — para PowerPoint
  1. Abra um documento no aplicativo online desejado
  2. Clique em Inserir → Suplementos
  3. Na janela de Suplementos do Office, clique em Gerenciar Meu Suplemento ▾
  4. Selecione Carregar Meu Suplemento
  5. Clique em Procurar… e selecione o arquivo manifest.xml
  6. Clique em Adicionar
Após carregar via Office Online, o add-in ficará disponível automaticamente no Word, Excel e PowerPoint Desktop (Windows e Mac) na sua conta Microsoft.

4 Iniciar o Claude Gateway

  1. Dê duplo-clique no Claude Gateway.vbs
  2. A interface gráfica abrirá com o painel de controle
  3. Clique em Iniciar para ativar o proxy
  4. O indicador mudará para ● ATIVO
  1. Abra o Terminal
  2. Execute o comando: 
    cd ~/.office-addin-dev-certs/.app && ./node/bin/node proxy-cors.js
  3. O proxy iniciará em https://localhost:8443

5 Usar o Claude no Office

O add-in funciona no Word, Excel e PowerPoint. O processo é o mesmo em todos:

  1. Abra o Word, Excel ou PowerPoint Desktop
  2. Vá em Suplementos → Mais Suplementos → Pesquisar por Claude
  3. O painel do Claude abrirá à direita
  4. Escolha a opção Cloud provider or gateway
  5. A Gateway URL deve ser a que o Claude Gateway gerou ao iniciar o proxy, basta copiar e colar
  6. Em Token, cole sua chave API do ZenMux
  7. Selecione o modelo e comece a usar!

? Solução de Problemas

"Could not retrieve the model list"

O proxy não está ativo. Inicie pelo Claude Gateway.vbs (Windows) ou via Terminal (Mac).

"The connection timed out. Please try again."

Rate limit do modelo gratuito. Aguarde ~30s e tente novamente. O proxy faz fallback automático.
Se o erro persistir, verifique o log de diagnóstico abaixo para identificar o problema.

Certificado não confiável

Repita o Passo 2 — instale o certificado como confiável.

Logs de diagnóstico

Get-Content "$env:USERPROFILE\.office-addin-dev-certs\.app\logs\proxy.log" -Tail 30
tail -30 ~/.office-addin-dev-certs/.app/logs/proxy.log
